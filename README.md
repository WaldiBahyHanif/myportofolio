# myportofolio
Judul Proyek: Portofolio Dinamis Berbasis Flask

Deskripsi: Aplikasi portofolio yang sedang Anda gunakan saat ini. Proyek ini dibangun dari nol (from scratch) menggunakan Python Flask sebagai backend dan MySQL sebagai database.

Fitur utamanya adalah panel admin backend (CRUD) yang terproteksi login. Dari panel admin, saya dapat dengan mudah meng-update bio dan foto profil, menambah atau menghapus skill (lengkap dengan ikon Font Awesome), serta meng-upload, meng-edit, dan menghapus proyek baru di galeri.
